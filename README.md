TUI-монитор процессов и системных ресурсов на C++.

### Сборка
Требуется CMake и компилятор с поддержкой C++17.

```bash
git clone https://github.com/saizzzy/Procsight.git
cd Procsight
cmake -S . -B build
cmake --build build
./build/bin/procsight
```
